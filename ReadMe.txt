Рад приветствовать!

В настоящем проекте реализовано тестовое задание.

Особенности проекта:
1.	Запас здоровья персонажей реализован через массив int.
	Массив int связан с массивом изображений очков здоровья (5 для игрока, 3 для противника).

2.	Движение персонажа по стенам возможно в случае "зацепа" bool isOnWall.
	Зацеп возможен когда игрок не находится на земле, имеет отрицательную скорость по высоте,
	 а его горизонтальное движение направлено всторону стены с которой имеется зацеп.
	Отскок от стены под 45 градусов с временным таймером ограничения движения.

3.	Настройка скорости движения, скольжения, таймеров и пр. выведены через инспектор объектов игрок и противник

4.	Движение противника от Триггеров EnemyStopper, от стен.
	Противник имеет детектор для разворота к игроку лицом и перехода в боевой режим.
